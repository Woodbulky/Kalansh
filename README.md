# Kalansh Website — Google Antigravity Project

## What this is
A complete showcase/brochure website for **Kalansh Udyog Nirmit** — an organic food brand from Kopargaon, Maharashtra. Static HTML + CSS + JS only. No backend, no ecommerce.

## How to Use in Google Antigravity

### Step 1 — Download images first
Open a terminal in this folder and run the curl script from `context/assets.md` to pull all product images from the live kalansh.in site.

### Step 2 — Open Antigravity
Open this entire folder as a project in Antigravity.

### Step 3 — Paste the master prompt
Open `MASTER_PROMPT.md`, copy the entire contents, and paste into the Antigravity chat.

### Step 4 — Let it plan
Use Antigravity's **Planning Mode** first. Review the plan before saying "proceed". This prevents surprises.

### Step 5 — Build phase by phase
Tell the agent to follow `tasks.md` in order. After each phase, review in the browser before moving to the next.

### Step 6 — Deploy
When all tasks are checked off, deploy to Netlify or Cloudflare Pages by dragging the folder.

## Key Design Decisions

| Decision | Choice | Why |
|----------|--------|-----|
| Fonts | Fraunces (display) + DM Sans (body) | Fraunces is organic and characterful; DM Sans is clean and readable |
| Primary colour | Deep Jaggery Brown `#4A2C0A` | Directly references the product — earthy and premium |
| Accent | Warm Saffron `#D47A1E` | Traditional Indian, warm, food-appetising |
| Background | Cream `#FAF6EE` | Natural, not clinical white |
| Radius | 20px on cards | Soft and friendly, not sharp corporate |
| No animations from 0 | Hardcoded stats | Prevents the embarrassing "0+" bug from the current site |
| No lorem ipsum | Real content in context/ | Prevents placeholder content going live |

## Important: No Ecommerce

This is a **showcase website only**. There are:
- No cart, no checkout, no payment gateway
- No prices displayed (except "starting from ₹5" as context)
- Product pages link to Amazon / Flipkart / Jio Mart for purchasing
- WhatsApp button for direct ordering enquiries
